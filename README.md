# MERN To-Do App

A full-stack to-do list application built using MongoDB, Express.js, React (Next.js 15), and Node.js. Each user has their own secure and private to-do list with date tracking, user-based CRUD operations, and full error handling.

## Features

- User registration and login (localStorage-based session)
- Create, read, update, delete (CRUD) operations on todos
- Each user has their own todo list
- Date and time displayed for each todo
- UI built with Tailwind CSS
- Full error handling (e.g. server or DB failure displays on UI)
- Responsive and clean user interface

## Tech Stack

| Layer      | Tech Used              |
|------------|------------------------|
| Frontend   | Next.js 15, React      |
| Backend    | Node.js, Express.js    |
| Database   | MongoDB                |
| Styling    | Tailwind CSS           |
| HTTP Client| Axios                  |
